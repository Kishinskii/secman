# Sync command

`secman-sync` is a **subprogram**, it's helps you to sync your passwords

## Commands

| Command | Alias | Work |
| :-----: | :---: | :--: |
| sync    |  sy   | start sync your .secman dir |
| clone   |  cn   | clone your .secman from private gh repo |
| push    |  ph   | push the new password |
| pull    |  pl   | pull secret/s |
