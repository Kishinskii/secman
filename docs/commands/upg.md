# Upgrade your secman if there's a new release.

Usage:
  `secman upg [flags]`

Aliases:
  **upg**, **upgrade**

Flags:
  -h, --help   help for upg

Note: `secman upg` & `secman-un` are only supported in **linux/mac**

## Linux/MacOS

> `update by upg command`

```sh
secman upg
```

> `by brew`

```sh
brew upgrade secman
```

## in Windows

```pwsh
scoop update secman
```
